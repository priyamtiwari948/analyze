About This Project

This project demonstrates a robust data processing pipeline using Python (3.11+), Pandas (2.3+), and GitHub Actions.It converts `data.xlsx` to `data.csv`, processes it with `execute.py` to fix a non-trivial error in data handling,and generates `result.json`. The entire process, including linting with Ruff, is automated and deployed via GitHub Pages.

Key Features

Robust Data Processing

Handles data cleaning and conversion with Pandas, fixing common errors like non-numeric values.

Automated CI/CD

Leverages GitHub Actions for linting, execution, and deployment on every push.

GitHub Pages Deployment

Automatically publishes the generated `result.json` as a static asset for easy access.
